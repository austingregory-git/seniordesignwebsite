ABOUT OUR PROJECT

Project Vision

Our goal is to develop a graph visualization tool that is:

  • Generalized so that it can be used across many domains
  • Fully featured and interactive
  • Integrated with Jupyter Notebook for ease of use
This tool could be used by researchers, students, and professionals to:
  • Understand and teach systems and processes more easily
  • Analyze graphs via algorithms
  • Share graphs and findings easily

Approach

Our approach involves building upon an existing tool: CHPG Developed by Ben Holland, an Iowa State alumnus The tool has a primitive implementation of tools for graph:

  • Visualization
  • Serialization
  • Analysis
We will expand upon this existing tool to add:
  • Visualization improvements
  • Interactivity
  • Generalization

Current Progress

Visualization

  • We have solved most issues related to visualization
  • We refactored the visualization code to more expandable and robust
Jupyter Integration
  • We have solved Jupyter integration on most systems
  • We have created a plan to implement a fully cross compatible solution
Graph Schema
  • We have gained an understanding of the problem domain
  • We have made tentative plans for implementing an improved and generalized graph schema

Future Plans

  1. Fix all remaining issues with visualization
  2. Fix integration with Jupyter
  3. Add graph interactivity features
    • Navigation
    • Analysis
  4. Create a generalized graph schema
    • Atlas graph serialization
    • Graph creation via API
